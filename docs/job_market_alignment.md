# Job Market Alignment: Canada Finance Analytics

## Target Roles

This project is designed for these Canadian roles:

- Data Analyst — Banking / Financial Services
- Business Intelligence Analyst
- Risk Analyst
- Fraud Analytics Analyst
- Customer Experience Analytics Analyst
- Compliance Analytics Analyst
- Junior Data Scientist / NLP Analyst
- AI / Automation Analyst

## Skill Mapping

| Job requirement | Project evidence |
|---|---|
| SQL / Python analytics | data cleaning, model training, API pipeline, reporting exports |
| Power BI / dashboarding | KPI-ready output tables and dashboard design pages |
| Machine learning | baseline, RNN, LSTM, Attention, BERT/RoBERTa comparison |
| NLP / text analytics | complaint classification, sentiment, topic/risk extraction |
| Azure / OpenAI | Azure OpenAI response agent with secure environment variables |
| Risk / compliance | escalation rules, SLA risk, model card, audit trail |
| Data quality | schema validation, missing narrative handling, model monitoring hooks |
| Business communication | executive README, dashboard storyboard, interview talking points |

## Resume Bullet Examples

- Built an end-to-end financial complaint intelligence platform using Python, NLP, FAISS RAG, Azure OpenAI, Streamlit, and FastAPI to automate complaint classification, escalation scoring, and response drafting.
- Compared TF-IDF Logistic Regression, RNN, LSTM, LSTM-Attention, BERT, and RoBERTa models for multi-class financial complaint classification using imbalanced-class evaluation metrics.
- Designed a policy-grounded RAG workflow to retrieve complaint-handling guidance and generate analyst-reviewable customer responses with human-escalation controls.
- Developed Power BI-ready complaint analytics outputs covering complaint volume, product risk, SLA exposure, sentiment trends, and model performance monitoring.
